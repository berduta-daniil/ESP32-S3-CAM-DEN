# GitHub Pages site

Статичний сайт для:

- перегляду відео з плати
- прослуховування аудіо з мікрофона плати
- відправки мікрофона телефона/ПК на динамік плати

Сайт працює через `MQTT over WSS`, тому йому не потрібен окремий локальний ПК.

## Як публікувати

Рекомендовано викладати цю папку в окрему гілку `gh-pages` і в GitHub Pages
вказувати source:

- branch: `gh-pages`
- folder: `/ (root)`

## Що вводити на сторінці

1. `Broker WSS URL` — рекомендовано `wss://broker.emqx.io:8084/mqtt`
2. `Device ID` — такий самий, як у `include/mqtt_bridge_config.h`
3. `Shared key` — такий самий, як у `include/mqtt_bridge_config.h`

Після цього:

- `Connect` підключає сторінку до MQTT broker
- `Start A/V` вмикає відео й звук з плати
- `Start talkback` шле мікрофон браузера на динамік плати

## Важливо

У рекомендованому режимі використовується публічний MQTT broker. Це зручно для
автономного запуску без VPS, але не підходить для чутливих даних.
